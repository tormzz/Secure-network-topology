# Secure-network-topology
VLAN + ACL secured enterprise (cisco packet tracer project)
# I am the network administrator for a small company. this topology is designed to secure the internal netwrok using VLAN Segmentation and Access Control Lists while applying the principle of Least Privilege.
# Network scenerio
 The organization consists of:
* IT Department IP( 192.168.10.0/24) 1PC, 1 SERVER
* Management IP (192.168.20.0/24) 2PCS
* Staff IP (192.168.30.0/24) 2PCS, 2 LAPTOPS
* External Network IP (10.10.10.0/24) 1PC
* 1 Router
* 1 Switch
